## Gaps Detail

### Gap 1 (RESOLVED): Power_Line F1 Diff Paper Inconsistency

**Location:** paper/main.tex, paper/data/paper_numbers.json
**Issue:** Previously, Section 3.1 used a single `\powerLineDiff = +0.394` (O4 value) in O3 context where the correct diff is +0.507.
**Status:** **RESOLVED.** Consistency checker confirms paper now has separate macros `\powerLineDiffOthree{+0.507}` and `\powerLineDiffOfour{+0.394}`, used correctly in their respective contexts. Grep confirms no residual confusion.

### Gap 2 (MINOR): N_total Discrepancy

**Location:** STATE.md (325,634) vs paper (325,632)
**Issue:** 2-sample difference, likely from gap-excluded samples in the temporal split.
**Fix:** Verify which count is authoritative and use consistently.

### Gap 3 (MINOR): Rare Threshold Protocol Override

**Location:** docs/experimental_protocol.md (N_rare=25) vs implementation (threshold=200)
**Issue:** Protocol says 25, implementation uses 200. The override is documented and defensible (4 classes give more robust rare-class average than 1), and Phase 3 uses the same threshold=200.
**Fix:** Add explicit note to protocol documenting the override.

### Gap 4 (INFO): Missing Phase Verifications

**Phases 1 and 3** lack VERIFICATION.md files.
- Phase 1 (data pipeline): Low risk — outputs consumed and validated by all downstream phases.
- Phase 3 (ViT evaluation): Medium risk — contains the core finding. However, the summary shows CNN re-evaluation with exact match, paired bootstrap with correct indices, and metric consistency checks. The forbidden proxy detection is well-documented.

### Gap 5 (INFO): CNN Accuracy Anchor Tension

**Phase 2:** Overall accuracy 91.81% falls below the [95%, 99%] contract threshold from Zevin et al. 2017. The explanation (O3 temporal split harder than O1/O2 random split) is plausible but no ablation was performed. This is documented as a known limitation and does not affect the decisive metric (rare-class F1 baseline).
