# Requirements: Paper Revision for CQG Submission (v1.1)

**Defined:** 2026-03-18
**Core Research Question:** Can a Vision Transformer trained on LIGO O3 spectrograms outperform Gravity Spy's CNN at classifying rare glitch morphologies, and can this improvement translate to better data quality for continuous gravitational wave searches?
**Milestone Goal:** Address 4 major + 5 minor issues from REFEREE-REPORT.md to make paper submission-ready.

## Primary Requirements

### Paper Text Revisions

- [ ] **TEXT-01**: Replace "class-morphology-dependent" with "class-dependent" throughout paper (abstract, §3.3, §4.1, conclusion); reframe Power_Line/Chirp examples as illustrative hypotheses, not established mechanisms
- [ ] **TEXT-02**: Add CW duty-cycle proxy validity statement in §2.4: "valid under the assumption that noise PSD is approximately stationary across the observation period"
- [ ] **TEXT-03**: Add Davis et al. (2021) citation in §2.4 when introducing the veto efficiency framework
- [ ] **TEXT-04**: Add limitation statement about single training run with no seed variance assessment; note per-class differences for rare classes may not be robust to training stochasticity
- [ ] **TEXT-05**: Add 2–3 sentences in §4.2 discussing asymmetric O4 degradation (ViT 7.4% vs CNN 1.7%) as evidence for the role of convolutional inductive bias in generalization

### Table/Figure Corrections

- [ ] **TABL-01**: Restructure CW Table 3: rename existing "Advantage" column to "F1 Advantage" and add separate "DC Advantage" column based on delta_DC sign; update table caption and all in-text references to Table 3
- [ ] **TABL-02**: Fix O4 macro-F1 rounding inconsistency — ensure Table 1 uses same LaTeX macros as running text, all rounded consistently to 3 decimal places

### New Computation

- [ ] **COMP-01**: Random-split ablation — retrain CNN baseline on same O3 data with standard random split (no temporal enforcement), report overall accuracy, and compare to temporal-split accuracy (91.81%) and published benchmarks (95–99%). Add results and discussion to §3 or §2.

### Statistical Analysis

- [ ] **STAT-01**: Compute statistical power analysis for rare-class macro-F1 comparison: minimum detectable effect size given n_test per class (Chirp=7, Wandering_Line=6, Helix=14, Light_Modulation=66), and probability of detecting the observed -0.062 difference
- [ ] **STAT-02**: Reframe rare-class narrative from "ViT underperforms on rare classes" to "insufficient evidence for ViT improvement on rare classes" — update abstract, §3.1, §4.1, and conclusion; present Chirp/Wandering_Line results as individually unreliable

## Follow-up Requirements

None — this is a terminal paper-polish milestone.
