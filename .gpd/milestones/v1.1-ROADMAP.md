## Phase Details

### Phase 6: Computation & Statistical Analysis

**Goal:** Produce the two new quantitative results the paper revision requires: (1) a random-split ablation confirming the temporal split explains the 5pp accuracy gap from published benchmarks, and (2) a power analysis quantifying the minimum detectable effect size for the rare-class comparison.

**Depends on:** Phase 5 (v1.0 paper draft, trained models, O3 dataset)
**Requirements:** COMP-01, STAT-01
**Contract Coverage:**
- Advances: REF-003 (accuracy gap ablation), REF-004 (power analysis for rare-class)
- Deliverables: Random-split CNN accuracy value with CI; power analysis table (MDE per rare class)
- Anchor coverage: ref-gravity-spy -- random-split accuracy compared to Zevin et al. 2017 published 95-99% range
- Forbidden proxies: Claiming temporal split explains the gap without running the ablation; claiming rare-class evidence is "insufficient" without computing statistical power

**Success Criteria** (what must be TRUE):

1. Random-split CNN accuracy is 3-8pp above temporal-split accuracy (91.81%), confirming that the split method -- not a pipeline bug -- explains the gap from published benchmarks (95-99%)
2. Power analysis reports minimum detectable effect size for each rare class given its test-set size (Chirp n=7, Wandering_Line n=6, Helix n=14, Light_Modulation n=66), demonstrating that the observed -0.062 rare-class macro-F1 difference is within the noise floor for the smallest classes
3. All new numerical values carry explicit uncertainty estimates or confidence intervals consistent with the v1.0 bootstrap methodology (10K resamples, seed=42)

## Backtracking Triggers

- Phase 6: If random-split accuracy is LOWER than temporal-split accuracy (91.81%), the accuracy gap explanation is wrong. Stop and investigate the training pipeline before proceeding to Phase 7.
- Phase 6: If power analysis shows sufficient power to detect the observed -0.062 effect, the rare-class regression may be real -- adjust Phase 7 reframing accordingly (genuine finding rather than "insufficient evidence").
- Phase 7: If reframing to "class-dependent" removes too much interpretive content, consider computing a morphological complexity proxy (spatial frequency entropy) as a fallback -- but only if the simpler reframing genuinely fails to carry the narrative.
