<objective>
Evaluate both O3-trained models (ViT-B/16 and ResNet-50 CNN) on O4 data to test whether the sample-efficiency threshold holds under distribution shift.

Purpose: The Phase 3 finding -- ViT outperforms CNN for classes with ~100+ training samples but underperforms for ultra-rare classes -- is the central paper claim. This plan tests whether the pattern is robust to O3->O4 distribution shift, which is essential for the claim's generalizability.

Output: O4 evaluation dataset with manifest, per-class F1 comparison table with O3 baselines, degradation analysis, Spearman correlation test for threshold persistence, scatter plot and degradation figures.
</objective>

    - Compute f1_diff_o4 = vit_f1_o4 - cnn_f1_o4 for each reliable class.
    - Spearman rank correlation: rho = spearmanr(n_train_o3, f1_diff_o4) using scipy.stats.spearmanr.
