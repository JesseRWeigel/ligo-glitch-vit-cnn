<objective>
Train a ResNet-50 CNN baseline on Gravity Spy O3 spectrograms with a modern training recipe, reproduce published CNN accuracy within 2%, and produce the per-class F1 breakdown that quantifies the rare-class performance gap.

Purpose: Establish the fair CNN performance floor that the ViT must beat in Phase 3. The CNN receives the SAME modern training recipe (AdamW, cosine LR, focal loss, class-balanced sampling) that will be applied to the ViT, ensuring any Phase 3 improvement is attributable to architecture, not training recipe.

Output:
- Trained ResNet-50 model checkpoint (best val macro-F1)
- Per-class F1/recall/precision table with bootstrap 95% CIs for all 23 classes
- Confusion matrix with rare-class highlighting
- Rare-class gap analysis document quantifying common vs rare F1 difference
- Training curves (loss, macro-F1 per epoch)
</objective>

    4. **src/losses/focal_loss.py** -- FocalLoss(nn.Module):
       - __init__(gamma=2.0, alpha=None, reduction='mean')
       - alpha: per-class weight tensor (sqrt-inverse weights from dataset)
       - Forward: FL(p_t) = -alpha_t * (1 - p_t)^gamma * log(p_t)
       - Use log_softmax for numerical stability: FL = -alpha_t * (1-p_t)^gamma * log_softmax[y]
       - Include unit test method: test_focal_loss() that verifies:
         * FL(p=0.9, gamma=2, alpha=0.25) = -0.25 * (0.1)^2 * log(0.9) = 0.000264 (within 1e-6)
         * FL(p=0.1, gamma=2, alpha=0.25) = -0.25 * (0.9)^2 * log(0.1) = 0.4669 (within 1e-4)
         * gamma=0 reduces to weighted cross-entropy (within 1e-6)
