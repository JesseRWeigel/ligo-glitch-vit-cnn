    2. **Abstract (~200 words):** MUST contain:
       - Problem: GW glitch classification with extreme class imbalance (23 classes, 11-120K training samples)
       - Method: ViT-B/16 vs ResNet-50v2 BiT with identical training recipe on 325K O3 Gravity Spy spectrograms
       - Central finding: ViT improves overall macro-F1 (0.723 vs 0.679, p<0.001) BUT underperforms on rare classes (0.241 vs 0.303)
       - Per-class insight: architecture preference is class-morphology-dependent, not a monotonic sample-size effect (Spearman rho=-0.034, p=0.88 on O4)
       - CW connection: at matched deadtime, veto efficiency is approximately equal; Power_Line F1 advantage (+0.394) is the main CW-relevant finding
       - Actionable message: per-class evaluation is essential; aggregate metrics mislead
       - DO NOT lead with overall accuracy

    3. **Introduction (~1.5 pages):**
       - GW detection, glitch contamination, detector characterization challenge
       - Gravity Spy project and CNN limitations (cite ref-gravity-spy: Zevin et al. 2017)
       - ViT motivation: global self-attention, transfer learning success in vision
       - Gap: no controlled per-class analysis of ViT vs CNN for imbalanced GW data
       - Our contribution: (1) class-morphology-dependent architecture preferences, (2) honest negative result on rare classes, (3) quantitative CW veto comparison
       - Cite: Zevin 2017, Zevin 2024, George 2018, Srivastava 2025, Wu 2024

    4. **Data and Methods (~3 pages):**
       - **Dataset:** 325,634 O3 Gravity Spy glitches, ml_confidence>0.9 filter, 23 classes (22 glitch + No_Glitch), 1 rare class (Chirp, 19 total)
       - **Preprocessing:** Pre-made Gravity Spy spectrograms from Zenodo, 224x224, ImageNet normalization
       - **Temporal split:** 70/15/15 by GPS time, >=60s gap, no temporal leakage
       - **CNN baseline:** ResNet-50v2 BiT (ImageNet-21k+1k pretrained), AdamW, cosine schedule, focal loss (gamma=2), class-balanced sampling, label smoothing 0.1
       - **ViT:** ViT-B/16 (AugReg ImageNet-21k+1k), IDENTICAL recipe except base LR and layer-wise LR decay (0.75)
       - **Fair comparison:** Same augmentation (horizontal flip, rotation +/-10deg, color jitter), same epochs, same evaluation
       - **Evaluation:** Macro-F1 (PRIMARY -- explicitly state why not accuracy), per-class F1 with 10K bootstrap CIs, paired percentile bootstrap for significance
       - **O4 validation:** 38,587 O4a glitches, same 23 classes, zero-shot evaluation (no O4 fine-tuning)
       - **CW veto:** Sample-count duty cycle proxy, matched-deadtime comparison, 7 CW-critical classes

    5. **Results (~3 pages):**
       - **Overall comparison (Table 1):** Present macro-F1, rare-class macro-F1, CIs, p-values for O3 and O4
       - **Per-class analysis (Figure 1 + Table 2):** The CORE of the paper. Show all 23 classes. Highlight: Power_Line (+0.394), Light_Modulation (+0.17), Chirp (-0.47). Discuss morphological reasons for per-class preferences.
       - **Threshold analysis (Figure 2):** Scatter of N_train vs F1_diff. Present Spearman correlation: O3 rho=-0.119, p=0.59; O4 rho=-0.034, p=0.88. State clearly: "The relationship between training set size and architecture preference is not a significant monotonic trend."
       - **Forbidden proxy lesson:** Explicitly note that overall macro-F1 improved significantly while rare-class macro-F1 declined. This demonstrates why per-class analysis matters.
       - **Confusion matrices (Figure 3):** Side-by-side, highlighting misclassification patterns for rare classes
       - **O4 generalization (Figure 4 + Table 1 O4 rows):** Both models degrade <20% (PASS). Per-class analysis of degradation patterns.
       - **CW analysis (Figure 5 + Table 3):** Matched-deadtime comparison (22.4%: ViT 0.745 vs CNN 0.735). Delta_DC = -0.051 [-0.054, -0.048] -- CNN preserves more livetime overall. Per-class: 2/7 CW-critical classes favor ViT (Power_Line, Violin_Mode), 5/7 favor CNN. Power_Line is the standout finding.

    6. **Discussion (~2 pages):**
       - **Why class-specific?** ViT's global attention helps for morphologically complex classes (Power_Line has multi-frequency structure); CNN's local features sufficient for simple blob-like classes (Blip). Rare classes lack data for ViT to learn attention patterns.
       - **O4 non-replication:** Distribution shift changes class morphologies; O4 has different detector noise. The per-class preferences are O3-calibrated observations, not universal laws. State this honestly.
       - **CW implications:** Operating-point-dependent. At the matched operating point, nearly equivalent. Power_Line is the most CW-relevant ViT advantage (60 Hz power line harmonics directly contaminate CW searches).
       - **The forbidden proxy lesson:** Overall accuracy (91.8% CNN, 93.4% ViT) would suggest ViT is strictly better. Per-class analysis reveals this is wrong for rare classes. Argue for per-class evaluation as standard practice.
       - **Comparison to prior work:** Srivastava 2025 reported 92.26% accuracy with ViT but no per-class analysis. Our work shows this aggregate metric hides important heterogeneity.
       - **Limitations:** Single-view (1.0s window), no data augmentation for ultra-rare, O3-specific threshold, duty cycle is coarse CW proxy, 23-class taxonomy may not cover O4 new morphologies

    7. **Future Work (~0.5 pages):**
       - Data augmentation for ultra-rare classes: cDVGAN (Lopez et al. 2024), contrastive pretraining
       - Multi-view attention fusion (Wu et al. 2024)

       - Class-specific confidence thresholds for CW vetoes
       - Frequency-resolved PSD analysis for more precise CW benefit quantification
