  <action>
    Create the release/ directory structure for both models.

    1. **Copy checkpoints:** Copy checkpoints/03-vit-rare-class/best_model.pt to release/checkpoints/vit_b16_gravityspy_o3.pt and checkpoints/02-cnn-baseline/best_model.pt to release/checkpoints/resnet50v2_gravityspy_o3.pt. Compute SHA-256 checksums and write to release/checkpoints/checksums.sha256.

    2. **preprocessing.py:** Extract eval_transforms() from src/data/transforms.py into a standalone file with NO training dependencies (no wandb, no dataloader). Must include:
       - IMAGENET_MEAN = [0.485, 0.456, 0.406], IMAGENET_STD = [0.229, 0.224, 0.225]
       - eval_transforms() using albumentations: Resize(224,224) + Normalize + ToTensorV2
       - A load_image() helper that reads a PNG, converts to RGB numpy array, applies eval_transforms

    3. **class_labels.json:** Copy from results/03-vit-rare-class/class_names.json. This is the index-to-class mapping for all 23 Gravity Spy classes.

    4. **model_config.json:** Create config specifying both models:
       - ViT: timm model_name="vit_base_patch16_224.augreg_in21k_ft_in1k", num_classes=23
       - CNN: timm model_name="resnetv2_50x1_bit.goog_in21k_ft_in1k", num_classes=23
       - Include pretrained source, input_size, normalization constants

    5. **inference.py:** Standalone CLI script:
       - Usage: python inference.py --model {vit,cnn} --image path/to/spectrogram.png [--top-k 3]
       - Loads model_config.json to construct architecture via timm.create_model(model_name, num_classes=23)
       - Loads state_dict from the appropriate checkpoint
       - Applies preprocessing.py eval_transforms to input image
       - Outputs: predicted class label, softmax probability, top-k predictions
       - Requires ONLY: torch, timm, albumentations, numpy, PIL -- no training code imports
       - Include ASSERT_CONVENTION comment at top: primary_metric=macro_f1, input_format=224x224_RGB_PNG_0to1

    6. **README.md (model card):** Create with these sections:
       - Model overview: ViT-B/16 and ResNet-50v2 BiT for Gravity Spy glitch classification
       - Training data: 227,943 O3 Gravity Spy spectrograms, 23 classes, temporal split
       - Performance summary table with macro-F1 (PRIMARY) and per-class highlights:
         - ViT macro-F1: 0.7230 [0.7018, 0.7413]
         - CNN macro-F1: 0.6786 [0.6598, 0.6944]
         - Key per-class: Power_Line ViT +0.394, Light_Modulation ViT +0.17, Chirp CNN +0.47
       - DO NOT lead with overall accuracy (fp-overall-accuracy). State it as secondary only.
       - Limitations: O3-trained, single-view (1.0s window), 23 classes, rare-class regression for ViT
       - O4 generalization: CNN macro-F1 0.6674, ViT 0.6695 (both <20% degradation)
       - Usage instructions with example command
       - Citation placeholder
       - License: MIT

    Read ALL numbers from existing result files (results/03-vit-rare-class/metrics.json, results/02-cnn-baseline/metrics.json, results/04-o4-validation/o4_metrics.json). Do NOT manually transcribe -- load and format programmatically where possible.
